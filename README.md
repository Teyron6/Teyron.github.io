# Сайт библиотеки Максима
## Сам сайт
На сайте находится [личная онлайн библиотека Максима](https://teyron6.github.io/Teyron.github.io/index.html), на данный момент в ней числятся 92 книги.

Сайт разделен на несколько страниц по 20 книг в каждой, передвигаться между страницами можно с помощью пагинаторов снизу и сверху страницы.

![Фото сайта](https://i.ibb.co.com/4ZJjVBjr/image.png)

Книги разделены на карточки, в картачке книги написана информация о ней, такая как название, автор, жанры, а так же ссылка на книгу

## Как скачать к себе на пк саму библиотеку
Для работы библиотеки нужны все файлы, кроме `render_website.py`, `README.md` и `.nojekyll`. После установки всех файлов к себе на пк, поместите все в отдельную директорию и откройте файл `index.html`. Вуаля, сайт работает у вас на пк

## Для продвинутых пользователей интересующихся проектом
Если ты продвинутый пользователь то я сомневаюсь что тебе необходимы подробные обьяснения, так что я вкратце расскажу о проекте :)

Начнем с самого интересного, с файла `render_website.py`. В нем была цель проекта и моего обучения на этом проекте, верстка на jinja2, а так же мой первый опыт работы с livereload. Он передает информацию файлу `template.html` и рендерит страницы, находящиеся в папке `pages`. Запускается скрипт коммандой `py render_website.py`, а выключается с помощью ctrl+c в терминале. В файле `requirements.txt` находятся необходимые для работы скрипта библиотеки(быстро скачать можно с помощью команды `pip install requirements.txt`)

Переходим к другим файлам и папкам. Файл `.nojekyll` отключает jekyll в github pages, файл `meta_data.json` хранит информацию о книгах, файл `template.html` хранит шаблон для рендера страниц сайта, файл `index.html` стартовая страница переходник. Теперь папки, папка `static` хранит статику, такую как bootstrap и favicon, папка `pages` хранит страницы сайта, папка `img` хранит обложки книг, ну а папка `books` соответственно сами книги.
